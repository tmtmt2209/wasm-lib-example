# `wasm-lib-example`
Example of javascript library package using [wasm-bindgen](https://github.com/rustwasm/wasm-bindgen).

## Requirements
- rust nightly toolchain
- target wasm32-unknown-unknown
- wasm-bindgen-cli
- node.js

## Install dependencies

```sh
$ npm install
```

## Build

```sh
$ npm run build
```

## Test

```sh
$ npm run test
```
